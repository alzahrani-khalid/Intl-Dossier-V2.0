## Trust Boundaries

| Boundary                                            | Description                                                                  |
| --------------------------------------------------- | ---------------------------------------------------------------------------- |
| intelligence_event pg_notify → backend alert worker | Trigger payload enters the trusted backend; clearance check is in the worker |
| Backend alert worker → external webhook URL         | Outbound HTTP; D-10 payload contract is the only content boundary            |
| Backend SMTP → on-prem relay                        | Outbound SMTP; SMTP_HOST env config gates this                               |

| T-70-12   | Information Disclosure | webhookAdapter payload leaking signal title/severity           | mitigate    | webhookAdapter only reads payload.genericLabel + payload.deepLink; D-10 contract tested in webhook-payload-contract.test.ts              |
