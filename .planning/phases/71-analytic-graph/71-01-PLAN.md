query_graph RPC signature (71-RESEARCH.md Pattern 1 — the contract the integration tests call):
query_graph(p_query_type TEXT, p_entity_id UUID, p_entity_id_2 UUID DEFAULT NULL, p_window_days INTEGER DEFAULT 90) RETURNS JSONB
p_query_type whitelist: 'forum_membership' | 'shared_committees' | 'engagement_chain' | 'shortest_path'
Returns jsonb_build_object with at least: { query_type, entity_id, nodes: [...], edges: [...], stats: {...} }

- forum_membership: nodes = forums/orgs the entity is member_of/participates_in
- shared_committees: nodes = working groups both entities belong to
- engagement_chain: nodes = engagements over p_window_days, ordered by start_date
- shortest_path: { path: uuid[], relationship_path: text[], path_length } (or empty when no path / above-clearance hop)

analytic-graph edge fn (71-PATTERNS.md — the HTTP contract the FE hook calls):
GET /functions/v1/analytic-graph?queryType=<type>&entityId=<uuid>&entityId2=<uuid?>&windowDays=<int?>
Authorization: Bearer <caller JWT>
Returns { nodes, edges, stats: { node_count, edge_count, query_time_ms, performance_warning } }
