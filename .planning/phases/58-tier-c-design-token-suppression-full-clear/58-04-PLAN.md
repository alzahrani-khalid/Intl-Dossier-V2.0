## Token mapping rules

Apply the token mapping table at `58-01-PLAN.md:110-127` verbatim (Pattern 1). Apply the Wave 2 additions in `58-02-PLAN.md` lines 80–110 verbatim.

Wave-4-specific decisions (5 semantic palette groups per RESEARCH section 5):

1. **Dossier-type palette (8 types)** — files: `DossierTypeGuide`, `DossierTypeSelector`, `CustomNodes` (per-node-type styling), and any sub-component that renders type badges (e.g., `ExpandableDossierCard` if it has type badges). **Honor `dossierTypeColors` map from `frontend/src/lib/semantic-colors.ts` verbatim — do not redefine the 8-type → ColorSet mapping inline.** Import: `import { dossierTypeColors } from '@/lib/semantic-colors'`.
2. **Recommendation score (high / medium / low)** — files: `DossierRecommendationCard`, `DossierRecommendationsPanel`, `FollowUpActions`, `AIFieldAssist`. Map: high → `text-success` / `bg-success/10`; medium → `text-warning` / `bg-warning/10`; low → `text-danger` / `bg-danger/10` (3 tiers fit cleanly on 3 status tokens — no opacity step needed).
3. **Engagement stage (planning / active / closed / cancelled)** — files: `EngagementsByStageCard`, `ConnectedAnchorsCard`. Map: planning → `text-info` / `bg-info/10`; active → `text-success` / `bg-success/10`; closed → `text-ink-mute` / `bg-muted`; cancelled → `text-danger` / `bg-danger/10`.
4. **Time-based recency (today / this-week / older)** — files: `DossierTimeline`, `CalendarEventsSection`, `FollowUpActions`, `StaffDirectory`. Map: today → `text-accent` / `bg-accent/10`; this-week → `text-info` / `bg-info/10`; older → `text-ink-mute` / `bg-muted`.
5. **Activity-status (completed / in-progress / pending / cancelled)** — files: `OutcomesSummary`, `FollowUpActions`, `CommitteeAssignments`, `TermHistory`. Map: completed → `text-success`; in-progress → `text-warning`; pending → `text-info`; cancelled → `text-ink-mute`. 4 tokens for 4 tiers — no opacity step.

For each blue-purple-collision file (14 files), decide each `text-blue-*` site by semantic context: link/CTA → `text-accent`; informational icon → `text-info`; selected outline → `text-primary`. For purple sites, follow Wave 2 WGMemberSuggestions precedent: `purple → accent`. Document any non-obvious decision inline as `// D-58-04-NN: <reason>`.
