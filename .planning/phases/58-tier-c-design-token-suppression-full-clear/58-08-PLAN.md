## Token mapping rules

Apply Pattern 1 (`58-01-PLAN.md:110-127`) and Wave 2 additions (`58-02-PLAN.md:80-120`) verbatim. Apply Pattern 5 semantic-context discrimination from `58-06-PLAN.md` where status palettes appear.

Wave-6C-specific decisions:

- **Page-level engagement stage palette** (EngagementDetailPage, OverviewTab, etc.) — match Wave 4 EngagementsByStageCard precedent: planning → `info`; active → `success`; closed → `ink-mute`; cancelled → `danger`.
- **Work-item status palette** (MyAssignments, WaitingQueue, WorkItemCard, WorkItemFiltersBar, AssignmentQueue, IntakeQueue, TicketDetail) — match unified work-item terminology per CLAUDE.md: `pending → info`; `in_progress → warning`; `review → accent`; `completed → success`; `cancelled → ink-mute`. For priority: `low → success`; `medium → warning`; `high → danger/80`; `urgent → danger`. (4-tier-on-3-token opacity-step per RiskList precedent.)
- **Admin route banners** (data-retention, field-permissions, ai-settings, ai-usage, approvals, preview-layouts) — admin-warning banners use `bg-warning/10 border-warning/30 text-warning`; admin-critical use `bg-danger/10 border-danger/30 text-danger`.
- **Help/documentation pages** (CommitmentsHelpPage, HelpPage) — informational callouts use `bg-info/10 border-info/30 text-info`. Code blocks / monospace surfaces use `bg-muted text-ink-mute`.
- **KPI cards** (productivity, intelligence) — KPI delta per Wave 5 KpiWidget precedent: up → `text-success`; down → `text-danger`; flat → `text-ink-mute`.
- **Activity statistics** (ActivityStatistics) — same activity-type palette as Wave 6B EnhancedActivityFeed: created → success; updated → info; deleted → danger.
- **Dossier-type badges in pages** (PersonsListPage, persons sub-pages, dossier-related route entries) — honor `dossierTypeColors` map; do NOT redefine the 8-type palette inline.
- **Blue-purple collisions (14 files)** — per-file decision: link/CTA → `text-accent`; informational icon → `text-info`; selected → `text-primary`. Purple → `accent`. Document non-obvious decisions inline as `// D-58-06C-NN`.
