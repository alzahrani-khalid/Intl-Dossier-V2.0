| Property | Value |

## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding in KPI cards |
| sm | 8px | Compact element spacing, Cmd+K result item padding |
| md | 16px | Default element spacing, widget internal padding |
| lg | 24px | Section padding inside dashboard zones, modal body padding |
| xl | 32px | Layout gaps between dashboard widget columns |
| 2xl | 48px | Major section breaks between absorption zones |
| 3xl | 64px | Page-level spacing (not used in this phase) |

Exceptions: Mini-graph container uses explicit 160px max-height (not a spacing token -- dimensional constraint for ReactFlow rendering). Touch targets remain 44x44px minimum per CLAUDE.md.

## Interaction Contracts

### Cmd+K Quick Switcher

| Interaction | Behavior |
|-------------|----------|
| Open | `Cmd+K` (Mac) / `Ctrl+K` (Win) from any page. Already wired via KeyboardShortcutProvider. |
| Close | `Escape` key or click outside dialog. |
| Navigate results | Arrow Up/Down cycles through all visible results across groups. |
| Select result | `Enter` navigates to entity or executes command. |
| Empty state (no input) | Show 5 most recent dossiers + 5 most-used commands (localStorage frequency tracking with fallback to preset popular commands). |
| Typing | Debounce 200ms (existing). Results replace empty state. Groups: Recent -> Entities (sub-grouped by type) -> Commands -> Search Results. |
| Result limits | 5 per entity type sub-group, 5 commands, 10 search results. Max ~50 visible items. |
| Mobile | Full-screen mode (existing behavior). |

### Mini-Graph in RelationshipSidebar

| Interaction | Behavior |
|-------------|----------|
| Render condition | Only render when sidebar `open === true`. Use `key={open}` to force remount on open. |
| Container | 160px max-height, full sidebar width minus 8px padding each side (mx-2). |
| Node hover | Show tooltip with entity name and type. |
| Node click | Navigate to dossier detail page. |
| "Expand Graph" click | Open FullScreenGraphModal (lazy-loaded). |
| Mobile | Mini-graph hidden (sidebar uses sheet/drawer pattern from Phase 12 -- graph not shown in sheet). "View Network" command available via Cmd+K instead. |

### Full-Screen Graph Modal

| Interaction | Behavior |
|-------------|----------|
| Open | Click "Expand Graph" in RelationshipSidebar, or "View Network" Cmd+K command. |
| Container | AdaptiveDialog at 80vh height, 0 padding. BottomSheet on mobile. |
| Controls | Zoom in/out buttons, fit-to-view, pan (drag). Filter by relationship tier. |
| Node interaction | Click to select (highlight connected edges), double-click to navigate to dossier. |
| Close | X button, Escape key, or click outside. |

### Dashboard KPI Widgets

| Interaction | Behavior |
|-------------|----------|
| Layout | 2-column grid on mobile (`grid-cols-2`), 4-column on sm+ (`sm:grid-cols-4`). Gap: 12px (`gap-3`). |
| KPI card | Static display: metric value + label + optional trend indicator (up/down arrow with percentage). |
| Click | Not clickable. KPIs are glanceable summary only (D-05). |
| Loading | Skeleton placeholder matching card dimensions. |
| Error | Muted error text inside zone, zone still renders. |
| Refresh | Automatic via TanStack Query stale-while-revalidate (5-minute stale time). |

### List View Export/Import Buttons

| Interaction | Behavior |
|-------------|----------|
| Location | Action bar area of list view headers (alongside existing filter/sort controls). |
| Export click | Opens ExportDialog (existing component) pre-filtered to current list view context. |
| Import click | Opens file picker dialog for CSV/JSON upload. |
| Touch target | Minimum 44x44px. Icon-only on mobile with tooltip, icon+label on sm+. |

## RTL Considerations

All new components in this phase MUST follow CLAUDE.md RTL rules:

| Concern | Implementation |
|---------|---------------|
| Logical properties only | `ms-*`, `me-*`, `ps-*`, `pe-*`, `text-start`, `text-end`. Never `ml-*`, `mr-*`, `pl-*`, `pr-*`, `text-left`, `text-right`. |
| Direction detection | `const { i18n } = useTranslation(); const isRTL = i18n.language === 'ar';` with `dir={isRTL ? 'rtl' : 'ltr'}` on containers. |
| Directional icons | Chevrons: `chevron-left` for "expand/enter", `chevron-right` for "back/return" in RTL. Arrow icons flip accordingly. |
| Cmd+K shortcut badges | Use `Cmd` not platform-specific symbol for consistency. Shortcut text does not flip. |
| Graph layout | ReactFlow uses absolute positioning -- unaffected by RTL. Node labels use `writingDirection: 'rtl'` for Arabic text. |
| KPI trend arrows | Trend up/down arrows are universal (not directional) -- no RTL flip needed. |
