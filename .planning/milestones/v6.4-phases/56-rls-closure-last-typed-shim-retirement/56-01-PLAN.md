| ------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------ |
| authenticated client → public.countries SELECT                            | Any signed-in user can read all rows (intended for ISO reference data).                                                                          |
| authenticated client (admin/editor role) → public.countries INSERT/UPDATE | Only users whose `public.users.role IN ('admin','editor')` may write. Role lookup happens inside the policy itself via subquery on `auth.uid()`. |
| authenticated client (admin role) → public.countries DELETE               | Only admins may delete.                                                                                                                          |
| service_role → all of the above                                           | Bypasses RLS entirely (Supabase convention). Test suite USES service_role to introspect `pg_policy`, NOT to modify data.                         |
