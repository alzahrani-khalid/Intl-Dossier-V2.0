# Migration Documentation

This directory contains migration guides and documentation for transitioning between technologies, data formats, or system architectures.

## Contents

### Package Manager Migration (PNPM)

- `PNPM_CHANGES_OVERVIEW.md` - Overview of PNPM migration changes
- `PNPM_ENFORCEMENT.md` - PNPM enforcement guidelines
- `PNPM_ENFORCEMENT_SUMMARY.md` - Summary of PNPM enforcement
- `PNPM_MIGRATION_COMPLETE.md` - PNPM migration completion report

### Geographic Data Migration

- `COUNTRY_MAP_MIGRATION.md` - Country map migration guide
- `GEOGRAPHIC_DATA_SOLUTION.md` - Geographic data solution documentation

## Adding New Migration Guides

When performing a significant migration, document it here with:

- `MIGRATION_NAME.md` - Detailed migration steps
- `MIGRATION_NAME_SUMMARY.md` - Summary of changes made

Include:

1. Before/after state
2. Step-by-step migration process
3. Rollback procedures
4. Verification steps
