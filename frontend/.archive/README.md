# Archived design documentation

These files were moved here on 2026-05-01 as part of adopting the
**IntelDossier Design System** at
`frontend/design-system/inteldossier_handoff_design/`.

They predate the prototype and use vocabulary that conflicts with the
current visual source of truth (Kibo UI, GASTAT green, mobile-first
breakpoints, shadcn defaults). Keeping them at the `frontend/` root
caused agents and reviewers to average across mutually-incompatible
guidance.

For the current canonical guidance, see:

1. `CLAUDE.md` → "Visual Design Source of Truth"
2. `frontend/DESIGN_SYSTEM.md` (slim pointer)
3. `frontend/design-system/inteldossier_handoff_design/README.md`

Files here are kept only for git-history/reference. Do not consult them
when designing or building new UI.
